### Run ###

Sous Linux :

### Installer Flask ###

apt install python3

pip install flask

### Installer SQLite ###

sudo apt install sqlite3

### Installer DOTENV ###

python-dotenv-0.21.0

For Javascript :
npm install --save-dev dotenv-webpack

### Installer NEWSAPI ###

npm install newsapi --save

### Installer les requirements ###

pip install -r requirements.txt

### Lancer le site ###

Via flask, executer :

export FLASK_DEBUG=1

flask run

Via python : 

python3 app.py
